# Sample Videos

This directory contains sample lifting videos for testing and demonstration.

## Usage

Place your sample videos here in MP4, AVI, or MOV format.

## Example Videos

- `squat_sample.mp4`: Sample squat video
- `bench_sample.mp4`: Sample bench press video
- `deadlift_sample.mp4`: Sample deadlift video

Note: Due to file size limitations, sample videos are not included in the repository.
You can find sample lifting videos online or record your own for testing.
